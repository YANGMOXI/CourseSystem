### 1、需求分析
    - 管理视图
        - 1.注册
        - 2.登录       
        - 3.创建学校
        - 4.创建课程（先选择学校）
        - 5.创建讲师
    
    - 学员视图
        - 1.注册
        - 2.登录       
        - 3.选择校区
        - 4.选择课程（选择学校>> 选择课程）
            - 双向绑定：学生选择课程，课程也选择学生
        - 5.查看分数
    
    - 讲师视图
        - 1.登录
        - 2.查看教授课程       
        - 3.选择讲授课程
        - 4.查看课程下的学生
        - 5.修改学生分数
    
### 2、程序的架构设计
    - 三层架构
        - 用户视图层
            - 用于与用户进行交互
            - 小的逻辑判断
        - 逻辑接口层
            - 核心业务逻辑处理
        - 数据处理层
            - 数据的 增、删、查、改
